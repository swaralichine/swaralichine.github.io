# Swarali Chine — Portfolio

Personal portfolio site for Swarali Chine, Senior Technical Support Engineer at Wasabi Technologies.

**Live:** [swaralichine.github.io](https://swaralichine.github.io)

---

## Tech Stack

- **Framework:** React 18 + TypeScript
- **Build tool:** Vite
- **Animations:** GSAP + ScrollTrigger
- **Smooth scroll:** Lenis
- **Marquee:** react-fast-marquee
- **Styling:** CSS (no framework)
- **Deployment:** GitHub Pages

---

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Building

```bash
npm run build
```

Output goes to `dist/`. Deploy by pushing to the `main` branch — GitHub Pages serves from root.

---

## Structure

```
src/
  components/
    Hero.tsx          # Landing section with marquee
    About.tsx         # About me + stats
    WhatIDo.tsx       # Areas of focus
    Career.tsx        # Timeline of work + education
    WasabiWork.tsx     # Wasabi project cards
    Work.tsx          # Side projects + research
    Awards.tsx        # Awards & recognition
    TechStack.tsx     # Tools, technologies, certifications
    Contact.tsx       # Contact section
    Navbar.tsx        # Navigation
  index.css           # Global styles + CSS variables
```
